# **App Name**: Tripify

## Core Features:

- Flight Search: Enhanced flight search interface with origin, destination, date, and number of adults.
- Flight Listings: Displays flight search results with detailed flight information.
- Destination Recommendations: Recommends popular travel destinations to inspire users.
- Airline Logos: Integration of airline logos to improve recognition in the flight listings.
- AI Travel Tips: Generative AI tool provides travel tips for the user's destination.

## Style Guidelines:

- Primary color: Sky blue (#42a5f5) to evoke trust and wanderlust.
- Background color: Light gray (#f5f5f5) for a clean and modern feel.
- Accent color: Orange (#ff9800) to highlight important actions.
- Headline font: 'Poppins', sans-serif, for headlines and titles.
- Body font: 'PT Sans', sans-serif, for readability in descriptions.
- Use detailed, visually appealing icons for navigation and key features.
- Subtle animations to enhance user experience during transitions.
